🎵 Telugu Music Genre Prediction & Playback System
📌 Project Overview

The Telugu Music Genre Prediction & Playback System is a desktop-based application concept developed using C and Windows API that predicts the genre of Telugu songs using feature-based classification and ANN-inspired logic.
The system analyzes musical attributes such as:
Danceability
Energy
Acousticness
Tempo
Speechiness
Popularity
Based on these features, it predicts genres such as:
Pop
Classical
Dance
Hip-Hop
Others
Additionally, the system integrates YouTube playback, allowing users to instantly listen to the predicted song.

🚀 Key Features

🎯 Feature-based music genre prediction
🧠 ANN-inspired weighted classification logic
🖥 Interactive Windows GUI
▶ Instant YouTube playback integration
📊 Real-time genre output display
🛠 Technologies Used
Language: C
Framework: Windows API
Concepts:
Artificial Neural Networks (ANN)
Rule-based classification
Feature engineering
